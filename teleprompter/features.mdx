## Intelligent Script Management

### Built-in File Manager

<Steps>
<Step title="Automatic Script Discovery">
  The app automatically scans your scripts directory and loads all compatible files
  
  ```bash
  # Default script locations:
  # Windows: %APPDATA%/teleprompter/scripts/
  # macOS: ~/Library/Application Support/teleprompter/scripts/
  # Linux: ~/.config/teleprompter/scripts/
  ```
  
  <Check>
  Supports .txt and .md file formats with automatic format detection
  </Check>
</Step>

<Step title="Real-Time Synchronization">
  Any changes made to script files externally are reflected instantly in the app
  
  - Edit in VS Code, Notepad, or any text editor
  - Changes appear in <100ms
  - No manual refresh needed
</Step>

<Step title="In-App Script Editor">
  Edit scripts directly within the teleprompter with a full-featured editor
  
  <Tabs>
  <Tab title="Quick Edit Mode">
    Edit scripts in a compact panel while previewing changes
  </Tab>
  
  <Tab title="Maximize Mode">
    Full-screen editing experience with syntax highlighting (for .md files)
  </Tab>
  </Tabs>
</Step>
</Steps>

### Focus Band

The focus band helps you maintain eye contact with the camera while reading.

<Frame>
  ```
  ┌─────────────────────────┐
  │                         │  ← Dimmed area
  │  ─────────────────────  │  ← Top of focus band
  │  HIGHLIGHTED TEXT HERE  │  ← Sharp, clear text
  │  ─────────────────────  │  ← Bottom of focus band
  │                         │  ← Dimmed area
  └─────────────────────────┘
  ```
</Frame>

**Customization Options:**
- Band height (1-5 lines of text)
- Band position (top, center, bottom)
- Dimming intensity (0-90%)
- Band color/highlight effect

<Warning>
Position the focus band at your camera's eye level for the most natural-looking presentation.
</Warning>

### Click-Through Mode

Work seamlessly with the teleprompter overlay visible but non-interactive.

<Steps>
<Step title="Enable Click-Through">
  Press `Ctrl+I` to make the window click-through
  
  - Window becomes transparent to mouse clicks
  - You can interact with apps beneath it
  - Teleprompter remains visible
</Step>

<Step title="Temporary Interaction">
  Hold `Ctrl` and click to interact with teleprompter controls
  
  ```typescript
  // Implementation concept
  if (event.ctrlKey && event.type === 'click') {
    enableInteraction();
    setTimeout(disableInteraction, 5000); // 5s timeout
  }
  ```
</Step>

<Step title="Disable Click-Through">
  Press `Ctrl+I` again to restore normal window interaction
</Step>
</Steps>
